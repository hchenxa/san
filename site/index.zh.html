功能特性

开放 设计

每一层都可以在运行时替换。无供应商锁定。

01

🔌 多 LLM 提供商

使用任何 LLM —— 从 Anthropic 到 Z.ai。通过 /model 随时切换。

02

🔍 可插拔搜索

Exa、Tavily、Brave 或 Serper。通过 /search 随时切换。

03

🧩 技能与扩展

技能、插件、MCP 服务器和子智能体——兼容 Claude Code，无需修改即可运行。

04

🎭 角色系统

Markdown 定义的角色——系统提示、技能和配置按用户或项目打包，通过 /persona 切换。

05

⚡ 原生性能

单一 Go 二进制文件。12 MB，~0.01 秒启动，零运行时依赖。

06

💾 会话持久化

会话自动保存、恢复和分支，支持自动上下文压缩。

07

✦ 自我进化 Lv.1 · 已上线

每经过几轮交互，后台审核者会将你最近的工作提炼为持久化记忆和可复用技能——智能体在你工作中不断升级，无需手动调优。Lv.1 现已上线；更深层级即将推出。

随处部署

一个二进制。随处运行。

一个 ~12 MB 的二进制文件，零运行时依赖——同一文件可在笔记本、云节点或 256 MB 边缘设备上无需修改即可运行。

💻 macOS 和 Linux 笔记本 ☁️ 云节点和虚拟机 📟 ARM 边缘设备 🐳 scratch 容器 ⚙️ CI 步骤和生命周期钩子 🔒 离线隔离环境

无需 Node.js，无需 Python，无需 node_modules。为什么小很重要及运行环境 →
