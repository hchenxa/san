快速开始

从安装到发出第一个提示词，只需几分钟。

1

安装

一条命令即可。随时重新运行以升级。

macOS / Linux

$ curl -fsSL https://raw.githubusercontent.com/genai-io/san/main/install.sh | bash

Windows (PowerShell)

> irm https://raw.githubusercontent.com/genai-io/san/main/install.ps1 | iex

更偏好 Go？go install github.com/genai-io/san/cmd/san@latest

2

启动并选择主题

运行 san 启动 San。首次启动时你需选择一个配色主题——选择后即可连接模型。

$ san
3

连接模型

输入 /model 打开模型设置，然后：

  1. 在右侧的提供商面板中粘贴你的 API 密钥——这将加载该提供商的所有可用模型。
  2. 等待提供商模型列表加载完成。
  3. 从模型列表中选择你想要的模型，开始编码。

随时通过 /model 切换提供商或模型。你的选择会保存到 ~/.san/providers.json。

准备就绪

输入 /help 探索斜杠命令，或深入了解文档。
